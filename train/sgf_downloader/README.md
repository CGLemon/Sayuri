# SGF DownLoader

The tool to download SGF file from katago server.

## Requirements

* requests

## Usage

| Parameter      | Type              | Description                 |
| :------------: | :---------------: | :---------------:           |
| -g, --game     | int               | Number of downloading games.|
| -d, --directory| string            | The saving directory.       |


    $ python3 main.py --game 1000 --dir saving
